What is this course?

This is a 3D Low Poly Art class, where you can learn to create a 3D work with a Low Poly style using the Blender software. There are 11 case study materials with different scenes and each material has its own characteristics and uniqueness, of course also accompanied by various techniques so that it makes you more creative in using Blender tools in the future.

In short, this class consists of several stages in each case study, starting from Modeling, Creating 3D Material Objects, Camera View Low Poly style, Setting Rendering Image, so you will really practice from the beginning to the final rendering stage.

In addition, the original Blender file belonging to the mentor is also provided, so that participants can download, and learn firsthand what the mentor makes.

Oh yeah, you can also consult about the material with the mentor directly through the telegram group, okay~

Why Should You Learn 3D Low Poly Art?

3D Low Poly Art is one of the styles that is developing in the 3D world, this style is widely liked because of the illustration of a 3D object with simple fields, so that the 3D illustration looks unique, even though it does not display realistic object modeling. In addition, this style tends to be easier to learn, and the manufacturing process can certainly be faster than creating 3D objects with a realistic approach, which requires a high level of detail.

This style is needed in many areas of the creative industry, for example for posters, illustrations for website/application needs, social media content, to advertising/commercial advertising needs, video advertisements (animation), and of course many more. In addition, this Low Poly 3D style is usually used in light game creation, because with a low number of polygons it speeds up the creation process and improves the game's performance.
